# Project Name: Ruff-Le Land üê∂

Ruffle Land is a **puppy-themed raffle web app** built using **React (Next.js) and JavaScript**, deployed on **Vercel**. The app allows users to explore and participate in raffles while ensuring **secure authentication** and data management.

## Features
- **Guest Users:** Browse existing raffles, view details and enter raffles.
- **Authenticated Users:** 
  - Create raffles üé´
  - Enter raffles üëÄ
  - Pick winners üèÖ
  - View personal raffle history üîç
- **SSO Authentication:** Ensures secure login for managing raffles.
- **MongoDB Integration:** Stores raffle-related data efficiently.

## Tech Stack
- **Frontend:** React (Next.js), JavaScript, Tailwind (CSS)
- **Backend:** Node.js/Next.js, MongoDB, Mongoose
- **Deployment:** Vercel
- **Auth:** Google Single Sign-On (SSO)
